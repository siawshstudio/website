# Siawsh Studio — Website

## Purpose
Main website and content system for Siawsh Studio.
Disciplines: branding, motion, interior design.

## Vision
Build a minimal, future-modern studio presence.
Each project is a case study, not just visuals.

## Website Sections
- Home — overview + philosophy
- Work — selected case studies
- Services — clear offerings
- Studio — story + process
- Blog — thinking, research, breakdowns
- Contact — direct and simple

## Content Rules
- Show process, not only final results
- Explain decisions briefly
- Keep language simple and human
